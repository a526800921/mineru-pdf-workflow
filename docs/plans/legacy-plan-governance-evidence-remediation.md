# 计划：历史计划治理证据补全

## 计划状态

- 状态：实施中
- 当前阶段：阶段 1：历史已完成计划证据补全（已完成）
- 最后更新：2026-07-15

本文档是历史治理证据补全的实施细节事实源。目标是补齐治理检查需要的结构化标题和可追溯验证说明，不改变历史计划的业务结论、代码、PDF 产物或数据状态。

## 目标

- 补全历史已完成计划缺少的 Step 0/验证方式和测试覆盖证据入口。
- 让 `plan-governance-cli check .` 不再因这些历史文档结构缺陷阻断当前 `pdf-table-repair` 的治理收尾。
- 明确补充证据是治理索引和当前仓库回归基线，不伪造历史行覆盖率百分比或业务验收事实。

## 非目标

- 不修改 `scripts/`、函数、类、CLI、PDF、segments、Markdown 业务产物或数据库。
- 不改写历史阶段结论，不把当前仓库回归结果冒充为历史阶段的专属覆盖率。
- 不删除历史计划、报告、ADR 或草案。
